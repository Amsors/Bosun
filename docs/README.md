# Bosun 用户文档

本目录是 Bosun 面向**最终用户**的产品文档：讲清楚「作为使用者，如何在网页上创建并使用托管的 coding agent 会话」。它随主仓库版本走。

> 面向开发者/贡献者的规范、设计与计划（spec / guide / record 及 PRD / techspec / plan）不在这里，而在 `dev_docs/`——一个独立 git 仓库，以 submodule 形式挂载。

## 目录

- [快速开始](getting-started.md) — 注册、创建第一个会话、打开终端。
- [常见问题](faq.md) — 计费、休眠恢复、BYOK 等。

> 产品尚在开发中，本目录内容随功能上线逐步补全。
